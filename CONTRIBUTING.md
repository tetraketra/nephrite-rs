This file is to be expanded as the project grows.

## Commit Messages
This project prefers Conventional Commits, https://www.conventionalcommits.org/en/v1.0.0/, but you can do whatever you want as long as each commit indicates the features changed, added, removed, etc.

## Development Workflow
This project uses the traditional Rust `cargo [build|run|test]` workflow style.

## Artificial Intelligence
This project is an AI-free repository, where "AI" refers to modern (as-of 2026) generative AI like ChatGPT/Codex, Gemini, or Claude. AI-generated code may not be committed, will not be merged, and should not be used. AI agents are forbidden from contributing. This does not apply to digital people in the possible future of true general-purpose AI.
